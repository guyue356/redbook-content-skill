**兼容矩阵**（rendering×palette 思路）：
- 柔焦/手绘调性（治愈、纯欲）→ 避开高饱和撞色，选 cream/macaron/pink-soft。
- 高级/极简 → 避开 macaron/pop，选 jewel/grey-luxe/mono。
- 科技/干货 → 避开 earth/vintage，选 tech/cool/editorial。

## 三、封面公式（Cover Formulas）

小红书封面是生死线。每篇选定一个主公式。构图用 SVG 实现（背景氛围 + 大字叠加）。

| 公式 | 结构 | 何时用 | 要点 |
|---|---|---|---|
| 大字报 | 居中/左对齐超大主标题 + 一行利益点 | 通用、最稳 | 主标题 ≤12 字，占画面 40%+，不放账号名 |
| 对比 | 左右/上下「前 vs 后」「误区 vs 正确」 | 方法类、避坑 | 中间分隔线，两侧同构不同色 |
| 清单 | 编号列表「1/2/3…」竖排 | 合集、盘点 | 每条一行，留白充足 |
| 数字冲击 | 巨大数字（如「3 个」「90%」）+ 小字注解 | 数据、结论 | 数字是主角，占 50% 面积 |
| Before-After | 双图/双区对比 | 改造、测评、效果 | 标注 BEFORE / AFTER |
| 人物金句 | 人物照/插画 + 叠加一句话 | 人设、情感、文艺 | 金句短、字大、留白 |
| 提问式 | 一个扎心问句 + 小字「答案在第 3 张」 | 痛点、引流 | 提问即钩子 |
| 测评打分 | 项目 + 星级/分数矩阵 | 好物、对比 | 视觉化评分，别纯文字 |

**封面必含两要素**（QA 强校验）：主标题（钩子）+ 副标题/利益点。缺任一 = 不合格。**不放账号名**。

## 四、候选方案写法范例

用户主题「打工人晨间习惯」，三方案：

```
[方案 A] 治愈系 — 治愈系 × cream × 大字报
  视觉：奶油底、圆角卡片、手绘太阳图标、柔焦
  色彩：奶油 #FBF7F0(60%) + 浅褐 #E8DFD3(30%) + 金 #C9A227(10%)
  情绪：温柔、松弛、像「小红书生活记录博主」

[方案 B] 干货风 — 干货风 × cool × 清单
  视觉：冷调底、左侧编号清单、右侧对比标注
  色彩：冷白 #EEF3F8(60%) + 蓝 #9DB8D4(30%) + 蓝 #2F6FB0(10%)
  情绪：清晰、可信、像「职场知识账号」

[方案 C] 高级感 — 高级感 × grey-luxe × 数字冲击
  视觉：大留白、巨号「3」、细线分隔、无装饰
  色彩：灰白 #F4F4F2(60%) + 浅灰 #CFCFCA(30%) + 近黑 #2B2B2B(10%)
  情绪：克制、高级、像「极简设计师账号」
```
